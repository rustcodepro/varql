# varql

- a graphql approach with embedded machine learning and database.
- algorithm with in the machinelearning rust file. 
- read the variant and the fasta file and the sam file from the assembled 
     variants and fits a logistic regression on the variants based on the quality. 

```
cargo build

```

```
./target/debug/varql 
Server running on http://127.0.0.1:4000/graphql
```
